## 一、设计原则

### 1.1 产品气质

整体视觉参考 **Reddit 社区风格**：白底卡片、浅灰页面背景、高对比正文、**橙红主色**强调行动点。目标是「信息密度适中、扫读友好、移动端优先」，而非重度装饰或拟物。

### 1.2 设计价值观

| 原则 | 说明 |
|------|------|
| **移动优先** | 默认按手机竖屏设计；`md`（768px）及以上再展开侧栏、多列布局 |
| **地图不抢戏** | 地图页保持内容区简洁；详情进抽屉/Sheet，避免全屏遮罩挡地图 |
| **复用壳组件** | Modal、PageState、SearchInput、Button 等统一从 `components/ui/` 或既有域组件引入 |
| **状态可感知** | 列表/详情须具备 loading / empty / error / 无权限（或 PR 说明 N/A） |
| **反馈及时** | 异步操作用 toast + 按钮 disabled；禁止无反馈的双击提交 |
| **Token 优先** | 优先使用 `tailwind.config.ts` / CSS 变量中的 reddit 色系，减少散落硬编码 |

### 1.3 禁止事项

- 在同一页面手搓第二套 Modal / Drawer 壳（用 `components/ui/modal.tsx` 或域内 vaul 模式）。
- 新建与 `SearchInput`、`EmptyState`、`PageLoading` 平行的搜索/空态/加载组件。
- 在尚未设计 dark mode 前引入深色主题分支。
- 对用户承诺未上线能力（如积分签到）——文案须与 PRD R44 一致。

### 2.8 层级（z-index）

**单一可信源：** `lib/ui/z-index.ts` + `tailwind.config.ts` + `app/globals.css` `:root`。

**App Portal 严格顺序：**

| Token 类名 | 值 | 元素 |
|------------|-----|------|
| `z-navbar` | 40 | 顶栏 |
| `z-sidebar-backdrop` | 44 | 管理端移动侧栏遮罩 |
| `z-sidebar` | 45 | 管理端侧栏 |
| `z-dropdown` / `z-navbar-dropdown` | 50 | 下拉、导航面板 |
| `z-tab-bar` | 55 | BottomTabBar |
| `z-map-control` | 60 | 地图 FAB、集市 Focus Bar |
| `z-map-banner` | 70 | 选点提示条 |
| `z-drawer-overlay` / `z-modal-overlay` | 100 | 抽屉/Modal 遮罩 |
| `z-drawer-content` / `z-modal-content` | 110 | 抽屉/Modal 内容 |
| `z-popover` / `z-tooltip-popover` | 120 | Select 下拉、表格操作菜单 |
| `z-toast` | 130 | Toast |
| `z-map-modal-overlay` | 200 | 地图页覆盖抽屉的 Modal 遮罩 |
| `z-map-modal-content` | 210 | 同上内容层 |

**AMap 画布层**（与 App 分离）：`lib/poi-map/z-index.ts` 中的 `AMAP_Z_INDEX`。

**约束：**

1. 禁止在 `app/`、`components/` 新增 `z-[数字]`；使用上表 Token 或 `zClass()` from `lib/ui/z-index.ts`。
2. 局部角标用父级 `relative` + 子级 `z-10`，勿占用 `z-map-control`。
3. 底部 fixed 组件：`z-index ≥ tabBar` 且 `bottom` 留 `4rem + safe-area`（全屏 Sheet 除外）。

POI / 集市移动抽屉 snap=0.35 时遮罩透明；snap=1 时 `bg-black/40` 可点关。

## 三、响应式与页面壳

### 3.1 断点

| 断点 | 含义 | 典型变化 |
|------|------|----------|
| 默认 | &lt; 768px | 单列、Bottom Sheet、底 Tab |
| `md:` | ≥ 768px | POI 右侧固定抽屉、管理端侧栏常驻、Tab 横排 icon+文字 |
| `wide-mobile` | 宽高比 ≥ 2:1 | `--mobile-content-max: min(100vw, 60vmin)` |
| `ultra-wide` | 宽高比 ≥ 21:9 | `--mobile-content-max: min(100vw, 50vmin)` |

### 3.2 全局壳（Root Layout）

```
┌─────────────────────────────┐
│ Navbar (fixed, h-16, z-40)  │
├─────────────────────────────┤
│ main (flex-1, overflow-y)   │
│   bg-gray-50 + pb for tab   │
├─────────────────────────────┤
│ BottomTabBar (z-55, safe)   │
└─────────────────────────────┘
```

- `html/body`：`100dvh`、`overflow-hidden`；滚动发生在 `main` 内。
- `themeColor`：`#FF4500`（PWA / 浏览器 UI）。
- `lang="zh-CN"`。

### 3.3 学生端页面模板

**标准列表/表单页**

```tsx
<div className="min-h-full bg-[#F6F7F8] px-4 py-6">
  <div className="mx-auto max-w-2xl space-y-4">
    {/* 可选：顶栏返回 + 标题 */}
    {/* 内容卡片 */}
  </div>
</div>
```

**带子页头部的详情页**（积分、反馈记录等）

- 顶部分割：3px 渐变条 `from-[#FF4500] to-[#FF6B3D]`。
- 白底 header + `border-b border-[#EDEFF1]`。
- 返回按钮：`rounded-full border ... text-[#FF4500] hover:bg-[#FFF7F5]`。

### 3.4 管理端页面模板

- 使用 `AdminLayout`：左侧导航 + 顶栏面包屑 + 主内容白底区。
- 控制台区块：`DashboardSection` — `rounded-xl border border-[#EDEFF1] border-t-[3px] border-t-[#FF4500]`。
- 数据概览：`StatCard` + `variant`（blue/green/amber/slate/emerald/orange）。
- 表格页：配合 `PaginationControls`（URL `?page=` 驱动）。

### 3.5 地图页特殊规则

- 地图占满 `main` 可视区；Navbar 下缘对齐 `.below-nav`。
- POI **桌面**：右侧 `max-w-md` 滑入抽屉 + 半透明遮罩。
- POI **移动**：vaul `snapPoints={[0.35, 1]}`，默认 35% 高度预览。
- Marker：24×24，`offset (-12, -12)`；样式类 `.flat-marker`（见 `globals.css`）。
- 选点模式：光标 `.cursor-map-pick`。

## 四、组件库与使用规范

### 4.1 基础 UI（`components/ui/`）

| 组件 | 文件 | 用途 |
|------|------|------|
| `Button` | `button.tsx` | `primary` / `secondary` / `ghost` / `danger`；支持 `loading` |
| `Input` / `Textarea` | `input.tsx` | 标准表单控件；`hasError` 错误态；配合 `FormField` |
| `ConfirmDialog` | `confirm-dialog.tsx` | `openConfirm()` 命令式确认；destructive 用 `variant="danger"` |
| `Modal` | `modal.tsx` | Portal 全屏遮罩；`containerClassName` 控宽；高层级用 `overlayClassName` |
| `Tabs` | `tabs.tsx` | Radix Tabs；灰底轨道 + 白底选中项 |
| `FormField` | `form-field.tsx` | label + 错误 + hint；配合 `FormFieldInputClass()` |
| `PaginationControls` | `pagination-controls.tsx` | 管理端分页；须 Suspense |
| `LoadingSpinner` | `loading-spinner.tsx` | 整页/区块加载（橙色环） |
| `Skeleton` | `skeleton.tsx` | 内联 shimmer 占位 |
| `PageLoading` / `PageEmpty` / `PageError` / `PageNoPermission` | `page-state.tsx` | 标准四态 |
| `DrawerRoot` / `DrawerContent` / `DrawerHandle` / `DrawerBody` | `drawer.tsx` | vaul Bottom Sheet；默认 snap `[0.35, 1]`、`z-drawer-*` |

> **注意**：移动 Bottom Sheet 优先 `components/ui/drawer.tsx`；POI 桌面侧栏仍用 Framer Motion。表单输入优先 `Input` / `Textarea`；`FormFieldInputClass()` 仍可用但已标记 deprecated。

### 4.2 共享业务组件

| 组件 | 路径 | 用途 |
|------|------|------|
| `SearchInput` | `components/shared/search-input.tsx` | 后台筛选；`variant="soft"` 用于前台；`clearable` |
| `EmptyState` | `components/empty-state.tsx` | 区块空态（图标圆底 + 可选 CTA） |
| `Navbar` | `components/navbar.tsx` | 全局顶栏 |
| `BottomTabBar` | `components/bottom-tab-bar.tsx` | 四 Tab；选中 `#FF4500` |
| `AuthGuard` | `components/auth-guard.tsx` | 需登录页包裹 |
| `ToastProvider` | `components/toast-provider.tsx` | 全局 toast |

### 4.3 图标

- 统一 **lucide-react**；默认描边图标，尺寸：
  - 内联：`h-4 w-4`
  - 按钮/Tab：`h-5 w-5`
  - EmptyState：`h-8 w-8`（容器内）
  - StatCard：`h-6 w-6`

### 4.4 按钮规范

| 类型 | 样式来源 | 场景 |
|------|----------|------|
| 主按钮 | `Button variant="primary"` 或 `bg-[#FF4500] text-white rounded-lg` | 提交、确认 |
| 次按钮 | `Button variant="secondary"` | 取消、次要操作 |
| 幽灵 | `Button variant="ghost"` | 工具栏 |
| 危险 | `Button variant="danger"` | 删除、封禁 |
| 文字/链接 | `text-[#FF4500] hover:underline` | 列表内轻操作 |
| Pill 返回 | `rounded-full border border-[#EDEFF1] px-3 py-1.5 text-sm` | 子页返回 |
| CTA 大块 | 渐变 `from-[#FF4500] to-[#E03E00]` + `active:scale-[0.98]` | 广场「发布帖子」 |

**交互**：`disabled:opacity-50`；异步时 `loading`（自动显示 Loader2）；主色按钮 hover 略加深。

### 4.5 表单规范

```tsx
<FormField label="邮箱" required error={errors.email}>
  <Input hasError={!!errors.email} type="email" ... />
</FormField>
```

带左侧图标的输入：页面用 `relative` 包裹，图标绝对定位 + `<Input className="pl-10" />`。

| 项 | 规范 |
|----|------|
| 标签 | `text-sm font-medium text-gray-700` |
| 必填 | 红色 `*` |
| 输入框 | `rounded-lg border px-4 py-2 text-sm` |
| 聚焦 | `focus:border-[#FF4500] focus:ring-2 focus:ring-[#FF4500]/20` |
| 错误 | 边框 `border-red-500` + `text-xs text-red-600` |
| 搜索防抖 | `useDebounce(value, 300)` |

### 4.6 Modal 规范

- 结构：`modal-header` / `modal-body` / `modal-footer`（`globals.css` 工具类）。
- 最大高度：`max-h-[min(85vh,calc(100vh-40px))]`。
- 正文区：`overflow-y-auto`；头尾 `flex-shrink-0`。
- 默认最大宽：`max-w-lg`（32rem）；可通过 `containerClassName` 调整。
- ESC / 点遮罩关闭（可配置）。

**二次确认（destructive 操作）**：使用 `openConfirm()`（`components/ui/confirm-dialog.tsx`），根 layout 已挂载 `ConfirmDialogProvider`。地图页已开 elevated Modal 时传 `elevation: "elevated"`。

```tsx
openConfirm({
  title: "删除商品",
  description: "确定要删除该商品吗？",
  variant: "danger",
  confirmText: "删除",
  elevation: "elevated", // 地图层可选
  onConfirm: async () => { /* 异步时确认按钮显示 loading */ },
});
```

禁止对 destructive 操作使用 `window.confirm` / `confirm()`。

### 4.7 Drawer / Sheet 规范

**桌面侧栏**（POI）：Framer Motion 自右滑入；宽 `max-w-md`；可下拉关闭。

**移动 Bottom Sheet**（`components/ui/drawer.tsx`，底层 vaul）：

- 默认 snap：`[0.35, 1]`；初始 `DEFAULT_DRAWER_SNAP = 0.35`
- 拖拽条：`DrawerHandle`（`w-12 h-1.5 bg-gray-300 rounded-full`）
- 高度：`h-[85dvh]`；圆角顶 `rounded-t-[14px]`；z-index：`z-drawer-overlay` / `z-drawer-content`
- 可滚动区：`DrawerBody` 带 `data-vaul-no-drag`
- 与地图共存：`modal={false}`（默认）；全屏 snap 时 `DrawerOverlay` 显示遮罩

### 4.8 卡片规范

**标准白卡片**

```
rounded-xl border border-[#EDEFF1] bg-white p-4 shadow-sm
hover:shadow-md transition-all  // 可点击卡片
```

**带品牌顶边**

```
rounded-xl border border-[#EDEFF1] border-t-[3px] border-t-[#FF4500] bg-white p-5 shadow-sm
```

**帖子/feed 卡片**

```
rounded-2xl border border-[#EDEFF1] bg-white p-4 shadow-sm hover:shadow-md
```

### 4.9 列表与分页

| 模式 | 组件/做法 | 场景 |
|------|-----------|------|
| URL 分页 | `PaginationControls` | 管理后台表格 |
| 加载更多 | 底部按钮 + `Loader2` | 广场帖子流 |
| 空列表 | `PageEmpty` 或内联空态卡片 | 须带引导动作 |
| 表格 | 管理端原生 `<table>` + 斑马纹/悬停（沿用现有 admin 页） |

### 4.10 反馈与通知

| 类型 | 实现 |
|------|------|
| 轻提示 | `react-hot-toast`；`position="top-center"`；`top: 80` 避开 Navbar |
| 成功 | 绿色 icon `#10b981` |
| 错误 | 红色 icon `#ef4444` |
| 进行中 | `toast.loading` + 固定 `id` 便于 dismiss |
| 角标 | 数字 badge 于 Tab、个人中心入口（红色小圆点/数字） |

### 4.11 加载与骨架

| 场景 | 用法 |
|------|------|
| 整页首次加载 | `PageLoading` 或 `Suspense` + `LoadingSpinner` |
| 区块刷新 | `LoadingSpinner className="py-12"` |
| 列表内联 | `Skeleton className="h-20 w-full"` |
| 按钮内 | `Loader2 className="h-4 w-4 animate-spin"` |
| Stat 数字 | `animate-pulse rounded bg-gray-200` |

**统一约定**：新页面优先 `LoadingSpinner`（品牌色环）；避免混用多种 unrelated spinner 样式。

## 五、场景化模式

### 5.1 底部 Tab 导航

| Tab | 路径 | 选中色 | 未选中 |
|-----|------|--------|--------|
| 地图 | `/` | `#FF4500` | `text-gray-500` |
| 管理* | `/admin` 或 `/super-admin` | 同上 | 同上 |
| 广场 | `/square` | 同上 | 同上 |
| 我的 | `/center` | 同上 | 同上 |

- 无管理权限时第二格占位 `pointer-events-none`。
- 未登录「我的」→ `/login?redirect=%2Fcenter`。
- `aria-current="page"` 于选中项。

### 5.2 个人中心入口列表

- 每行：左图标（圆角底 `#FFE5DD` + 橙色 icon）+ 标签 + 可选 badge + `ChevronRight`。
- 容器：白卡片分段；logout 单独 destructive 样式。

### 5.3 广场信息流

- 顶部渐变 CTA（已登录）。
- 三列快捷入口 grid。
- 帖子卡：头像圆、`Globe` 校际标、图片 grid、互动区预留。
- 空态：圆角图标容器 + 一句主文案 + 一句副文案。

### 5.4 管理端 Stat + 快捷入口

- `StatCard`：`variant="orange"` 用于主指标；`urgent` 时 amber 边框 + 「点击立即处理」。
- `CommandShortcuts`：与 DashboardSection 相同顶边强调。

### 5.5 认证页（登录/注册）

- 居中表单卡片；错误用 `AlertCircle` + 红底提示条。
- 须 `Suspense` 包裹 `useSearchParams` 内容。
- 全屏 fallback：`LoadingSpinner min-h-screen`。

### 5.6 地图 FAB 与浮层

- 集市等 FAB 遵循现有 `fab-buttons` / 地图页布局，不遮挡 BottomTab 与定位按钮。
- z-index 低于 Modal，高于地图控件。

## 六、动效与微交互

| 场景 | 实现 | 参数参考 |
|------|------|----------|
| Modal 入场 | CSS `modal-overlay-fade` / `modal-container-in` | 0.2s ~ 0.25s ease-out |
| 桌面抽屉 | Framer Motion `x: 100% → 0` | spring damping 25 |
| Sheet snap | vaul 内置 | 0.35 / 1 |
| 按钮点击 | `active:scale-[0.98]` | 可点击卡片 |
| 留言高亮 | `animate-comment-highlight` | 1.2s |
| Marker 脉冲 | `.highlight-pulse` / `.live-pulse-dot` | 1.5s infinite |
| Skeleton | `.shimmer` | 1.5s |

**原则**：动效短、有目的；地图场景避免长时全屏动画。

## 七、无障碍与国际化

### 7.1 无障碍

- 全局 `:focus-visible`：白环 + 橙色外环（`globals.css`）。
-  icon-only 按钮须 `aria-label`（如 SearchInput 清除）。
- Modal：`role="dialog"`、`aria-modal="true"`。
- BottomTab：`aria-label="底部主导航"`。
- 装饰性图标：`aria-hidden="true"`。

### 7.2 文案与语言

- 界面默认**简体中文**。
- 时间显示：优先 `formatRelativeTime` 等现有工具。
- 数字：统计用 `tabular-nums` 对齐。
