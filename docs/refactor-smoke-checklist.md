# 重构回归冒烟清单（每 PR 勾选）

在合并涉及鉴权、学校、地图、集市、POI 的 PR 前，在**测试环境**按模块抽样执行。

## 门禁命令

- `npx tsc --noEmit`
- `npx next lint`（若本地/CI 已启用）

## 认证与会话

- [ ] 登录成功 / 登出后跳转登录页
- [ ] 会话过期或停用账号：访问受保护页应被拦或清空会话

## 首页 / 地图 / POI

- [ ] 选校、地图加载、点击 POI 打开抽屉
- [ ] 已登录：留言、举报（未登录应提示登录）

## 集市（抽样）

- [ ] 打开集市、浏览商品；卖家/买家关键操作任选一条路径

## 超级管理员

- [ ] 学校列表加载
- [ ] **删除学校**（仅测试库/可废弃数据）

## 校级管理员

- [ ] 校区管理页打开、列表加载（抽样保存若本次 PR 触及）

## 登录 / 注册

- [ ] 注册页 / 登录页：学校列表可加载
