## 开发常用终端命令速查

下面的命令默认在项目根目录 `D:\Projectdemo` 下执行。

---

### 一、数据库相关

- **同步 Prisma Schema 到数据库**

```bash
npm run db:push
```

- **生成 Prisma Client（修改 schema 后）**

```bash
npm run db:generate
```

- **打开 Prisma Studio（可视化查看/编辑数据）**

```bash
npm run db:studio
```

- **执行种子脚本（初始化/生成测试账号与基础数据）**

```bash
npm run seed
```

> 说明：`prisma/seed.ts` 中包含超级管理员、测试学校、POI 等初始化逻辑，可根据需要调整后再运行。

---

### 二、本地开发与构建

- **启动开发服务器（热更新）**

```bash
npm run dev
```

- **生产构建**

```bash
npm run build
```

- **启动生产环境服务器（基于 build 结果）**

```bash
npm run start
```

---

### 三、代码质量与调试

- **运行 ESLint 检查**

```bash
npm run lint
```

---

### 四、实用组合命令（建议手动逐条执行）

- **首次拉取项目后的一次性初始化流程（依次执行）**

```bash
npm install
npm run db:push
npm run seed
npm run dev
```

---

后续如果新增脚本（例如一键重置某校数据、一键导入 POI 批量数据），也建议统一写在 `package.json` 的 `scripts` 中，并在本文件补充说明。+
